# 5학년 1학기

## 1. 과학자는 어떻게 탐구할까요?

### 1.1 과학탐구

#### 1.1.1 탐구

* 스스로 탐구 문제를 정하고, 실험을 설계하여 수행하고, 결과를 정리하고 해석하여  결론을 내리는 과정이 포함된다.

#### 1.1.2 탐구 문제 정하기

* 문제인식 : 우리 주변의 자연 현상을 관찰하고, 탐구할 문제를 찾아 명확하게 나타내는 것을 말한다.
* 문제 인식 후 탐구 문제를 정하기 : 탐구하고 싶은 내용을 분명하게 드러내고, 좁고 구체적으로 범위를 정해야 한다.

#### 1.1.3 실험 계획 세우기

* 변인통제 : 실험에서 다르게 해야 할 조건과 같게 해야 할 조건을 확인하고 통제하는 것
* 변인통제 설정 후 관찰/측정 대상 확인하기
* 준비물, 실험과정, 모둠 구성원의 역할 등을 정하기

#### 1.1.4 실험하기

* 변인통제에 유의하면서 계획한 과정에 따라 실행한다.
* 관찰하거나 측정하려고 했던 것을 생각하면서 결과를 기록한다.
* 실험 결과를 있는 그대로 기록한다.
* 실험 결과가 예상과 다르더라도 고치거나 빼지 않는다.

#### 1.1.5 실험 결과 정리하고 해석하기

* 자료 변환 : 실험 결과를 표나 그래프의 형태로 바꾸어 나타내는 것
* 자료 해석 : 변환한 표나 그래프로 실험 결과를 통해 알 수 있는 점을 생각하고, 자료 사이의 관계나 규칙을 찾아낸다. 

#### 1.1.6 결론 내리기

* 결론 도출 : 실험 결과에서 결론을 이끌어 내는 과정
* 결론 도출 후 궁금증에서 새로운 탐구를 시작할 수 있다.

## 2. 온도와 열

### 2.1 온도와 열의 이동

#### 2.1.1 온도

* 온도 : 물질의 차갑거나 따뜻한 정도
* 온도를 나타내는 단위 :  °C (섭씨도, Celsius)
* 온도를 측정하는 기구 : 온도계

#### 2.1.2 온도계 사용법

* 귀 체온계
  * 체온을 측정할 때 사용한다.
  * 요즘은 귀 체온계보다 비접촉식 체온계를 사용함

* 적외선 온도계
  * 적외선 온도계로 측정하려는 물질의 표면을 겨누고 측정 버튼을 눌러 온도를 확인한다.
  * 액체 온도 측정시 유의사항 : 물질의 표면 온도를 측정하는 방식이기 때문에 액체 물질의 온도 측정은 정확성이 떨어질 수 있다.

* 알코올 온도계
  * 용도 : 액체나 기체의 온도를 측정할 때 사용한다.
  * 구조 : 고리, 몸체, 액체샘. 최근에는 색소를 넣은 기름으로 알코올 액체를 대신하기도 한다.
  * 측정 방법 : 액체의 움직임이 멈추면 액체 기둥의 끝이 닿는 부분의 눈금을 읽어 온도를 측정한다.
  * 액체 온도 측정시 유의사항 : 액체샘을 액체 속에 잠기도록 넣는다.

#### 2.1.3 물질의 온도 측정

* 용도(쓰임새)에 맞게 온도계를 적절한 방법으로 사용한다.
  * 체온 측정 - 귀체온계, 비접촉식체온계
  * 나무 그늘의 흙, 벽 등 고체 물질/표면의 온도 - 적외선 온도계
  * 방의 기온, 바깥의 기온 - 알코올 온도계
    * 주의사항 : 지열을 배제해야 하기 때문에 땅으로부터 1.5m 정도의 높이에서 측정해야 한다. ex) 백엽상
  * 냄비 속 기름 온도, 연못 속 물의 온도 - 알코올 온도계

#### 2.1.4 열의 이동

* 두 물질 간 열의 이동
  * 고온 → 저온
  * 온도가 다른 두 물질이 접촉하면 따뜻한 물질의 온도는 점점 낮아지고 차가운 물질의 온도는 점점 높아져서, 시간이 지나면 두 물질의 온도는 같아진다.
    * 단, 온도 차에 따라 열전도 속도가 정해져있다.
    * 예시 : 고기가 팬에서 익는다.
